# project_0
Project 0 Vertex AI
This project show case my ability to use BigQuery for data processing and exploratory data analysis and the Vertex AI platform to train and deploy a custom TensorFlow Regressor model to predict customer lifetime value (CLV) in *pharmaceutical environment*. 

## Key Features
- Interactive dashboard built using Tableau
- Insights on sales trends, customer segmentation, and revenue drivers

## Files
- `data/`: Contains the sales dataset used for the analysis.
- `code/`: Python scripts and Jupyter Notebook for data cleaning and preparation.
- `dashboard/`: Tableau file for the dashboard.

## Usage
1. Download the repository.
2. Open `customer_sales_dashboard.twbx` in Tableau Desktop or Tableau Reader.
3. Explore the interactive dashboard.

## About the Dataset
The dataset contains sales transactions for 2023, with the following fields:
- `Date`: Date of the transaction
- `Customer ID`: Unique identifier for the customer
- `Product`: Product purchased
- `Sales`: Sales amount in USD
